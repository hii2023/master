# Master — Projects Index

A personal portfolio hub that **auto-aggregates all GitHub repositories** via the GitHub REST API and displays them in a clean, paginated grid. Built by Altaf Simavatwala.

## What It Does

Pulls all public repos from `api.github.com/users/hii2023/repos`, sorts them by last updated, and renders them as a browsable project index — with the ability to add custom links manually.

## Key Features

- **Live GitHub repo feed** — auto-fetched, no manual updates needed
- **Sorted by last updated** — most recent work always surfaces first
- **Paginated grid** — handles large numbers of repos cleanly
- **Add Link / Page modal** — manually pin custom projects or external URLs
- **Hero section** — "Things I've Built" intro
- **Zero dependencies** — vanilla HTML/CSS/JS, no framework

## Tech Stack

- HTML, CSS, Vanilla JavaScript
- GitHub REST API (`api.github.com`)
